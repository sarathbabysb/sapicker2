Sapicker Demo & Documentation

Sapicker is a custom select dropdown that supports images, dynamic options, and reset functionality. Below is a live demo and copy-ready code examples.

1. Basic Picker

<div class="c-flag-sapicker">
  <select class="sapicker" id="picker-one">
    <option value="" selected>Select Option</option>
    <option value="in" data-thumbnail="./images/in.png">India</option>
    <option value="tu" data-thumbnail="./images/tr.png">Türkçe</option>
  </select>
</div>

2. Dynamic Picker

<select class="sapicker" id="dynamic-data">
  <option value="" selected>Select Option</option>
</select>

<script>
 function loadDynamic() {
  const newData = [
   { value: 'tl', text: 'Timor-Leste', image: './images/tl.png' },
   { value: 'tr', text: 'Türkçe', image: './images/tr.png' }
  ];
  Sapicker.addOptions('dynamic-data', newData, 'tu');
 }

 function resetDynamic() {
  Sapicker.reset('dynamic-data');
 }
</script>

3. Css & JavaScript Include

<style src="./css/sapicker.css"></style>
<script src="./js/sapicker.js"></script>

4. JavaScript Include

Include the Sapicker script and initialize:

<script>
 window.addEventListener('DOMContentLoaded', () => {
  Sapicker.init();
 });
</script>
